# STATIC City Assets

Owner-only city prototype assets should land in the category folders below before being wired into the runtime.

- `characters/`: playable Entity rigs and NPC rigs.
- `vehicles/`: driveable or cinematic vehicle models.
- `venues/`: streamed city chunks, interiors, rooftops, clubs, stores, and landmarks.
- `props/`: drinks, cigars, clothing racks, jewelry cases, phones, signs, and status items.
- `audio/`: rights-safe ambient loops, station cues, and UI stingers.
- `materials/`: shared textures, decals, light masks, and signage atlases.

Key production docs:

- `ASSET_SOURCING.md`: sourcing rules and generation prompts.
- `ASSET_MANIFEST.md`: production target checklist with license/source state.
- `BLENDER_PREP_GUIDE.md`: what this MacBook can safely do with GLB assets.
- `characters/METAHUMAN_PIPELINE.md`: human/humanoid character path.
- `venues/HIDDEN_WALKING_STREET_PROMPT.md`: hidden mature-nightlife venue chain prompt.
