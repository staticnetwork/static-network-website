# STATIC Visual Asset Shot List

This file is the current generation guide for upgrading the STATIC public site and preparing the later Unreal Engine city build. The rule: every asset should feel like the same original world, not a collage of unrelated AI images.

## Visual Canon

- Original luxury entertainment megacity at night.
- Wet reflective streets, black chrome, warm gold, ultraviolet, deep blue, neon magenta, and premium white signage.
- Dense city energy: fans, creators, performers, security, valet, cars, screens, rooftop parties, arenas, studios, shopping, beach/marina life, and hidden nightlife.
- The public site should feel intentionally motion-designed, not like static
  images inside a template. Use premium After Effects-style motion language:
  cinematic reveals, transmission overlays, holographic HUDs, particle energy,
  billboard loops, scanline glints, camera drift, UI boot sequences, map pulses,
  light sweeps, rain/reflection shimmer, and tasteful glitch/signal treatment.
- Motion assets must be owned, generated, custom-made, or properly licensed for
  commercial use. Do not download or ship unlicensed YouTube clips, ripped game
  footage, copied brand graphics, template previews, watermarked stock, or
  assets with unclear rights.
- No literal copies of real landmarks, real brands, GTA, Ready Player One, Fortnite, 2K, clubs, teams, logos, or celebrities.
- STATIC logo marks in generated art are concept placeholders only until the official logo system is finalized.
- AI-generated STATIC marks are not authoritative branding. Wrong marks in hero art should be treated as temporary concept artifacts and later replaced, masked, or composited with the official logo system.
- Use landscape 16:9 for web heroes and short cinematic loops. Use portrait 9:16 only for mobile-specific cutdowns.

## District Visual Rules

- No two districts should look like the same street with different signs.
- Use real-life photographic language first: real camera, real crowd, imperfect faces, motion blur, wet reflections, scuffed surfaces, believable security, actual physical clutter.
- Use Unreal/engine language only for internal planning, not as the main generation style for public route stills.
- Avoid overusing purple. STATIC City can include gold, emerald, cyan, red, amber, white, deep blue, magenta, and realistic city color spill.
- Every district needs one strong identifying behavior: arriving, shopping, racing, creating, broadcasting, partying, escaping, relaxing, competing, or earning status.

## Animation Targets

Short loops are more valuable than stills when they are clean and consistent.

- Arrival District: 15-20 second establishing loop, minimal hard cuts.
- Route travel: 5-8 second car/tram/elevator transition clips per major venue.
- Billboard loops: 3-5 second screen-only overlays for Signals, Channels, Radio, PLAY, Marketplace, Live, Studio.
- Ambient loops: crowd motion, car reflections, skyline lights, rain shimmer, drones, scanlines.
- Mobile loops: portrait crop with blurred/extended edges, not a square image pasted into mobile.
- Motion graphics overlay pack: licensed/generated alpha-friendly loops for
  boot-up reveals, lens sweeps, neon particles, waveform pulses, signal noise,
  map-region pulses, creator-card reveals, CTA glow states, lower-third labels,
  countdowns, access scans, and cinematic section transitions.
- Performance rule: use compressed web video, Lottie, CSS/WebGL, sprites, or
  canvas where appropriate; respect reduced-motion settings; never let the
  billion-dollar feel turn into slow page loads.
